FAQ

Common questions

Everything skeptical writers ask before they install.

Do I need an account?

No. GramAI never asks for your email. There is no login, no signup, no account.

Does it work on company/work computers?

Yes. Local mode doesn't contact any external server, so corporate firewalls and VPNs aren't an issue.

What happens if Ollama crashes?

We detect this and show you a one-click restart prompt inside the extension. No data is lost.

Is my Pro Cloud data stored?

No. Requests are processed and immediately discarded. We don't log, train on, or store any user text.

How is this different from just using ChatGPT?

GramAI is always one click away on any webpage. ChatGPT requires switching tabs, copying text, pasting, reading a long response, and copying back. GramAI does it inline in 2 seconds.

Can I use my own AI model?

Yes (Free/local). You choose which Ollama model runs — Llama, Qwen, Mistral, Phi, or any other.

What if I want to cancel Pro?

One click in your account dashboard. No dark patterns, no "call to cancel."

Install GramAI

GramAI isn't on the Chrome Web Store yet. Install from GitHub in under a minute:

  1. Download or clone the repo from GitHub
  2. Open chrome://extensions (or your browser's extensions page)
  3. Enable Developer mode
  4. Click Load unpacked and select the extension/ folder
